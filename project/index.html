Firebase Property App

Transparency for Real Estate Pricing
List, review, and analyze properties with community trust.
  • 🔑 Login
  • 📝 Register
  • 📊 Dashboard
  • 🏠 Properties (CRUD)
  • 🗣️ Reports (Community)

✨ Key Features

  • Authentication — Secure email/password login & registration
  • Property Listings — Add, view, and manage properties
  • Community Reports — Submit, edit, and view price/trust reports
  • Aggregated Insights — Average price/trust per property, global dashboard
  • Dashboard — All properties, reports, and metrics at a glance
  • Modern UI — Responsive, modular, and user-friendly
  • Firebase Hosting — Fast, secure, and scalable

How it Works

  1. Register or log in to your account
  2. Add a property with price and details
  3. View all properties and select one to see community reports
  4. Submit your own report on fair price and trust
  5. See live metrics and insights on the dashboard
